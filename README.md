# VillagerTradeEditor
An APP can make a transaction of Minecraft`s villagers conviniently(Just output mojang`s json)
This software serves only Minecraft Bedrock Edition. The output files are in Mojang's JSON format, which need to be converted between JSON and NBT using other tools, resulting in mcstructure files (NaroX is recommended for this purpose).

这个软件仅仅服务于Minecraft bedrock，输出的文件是mojang的json，需要使用别的工具进行json和nbt之间的转换，转换结果一般为mcstructure（这里推荐使用NaroX）
